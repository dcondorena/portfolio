# Portfolio — Diego Condorena

Personal portfolio website for Diego Condorena, Senior Software Engineer.

**Live site:** https://dcondorena.github.io/portfolio/

## Stack

Plain HTML and CSS — no build step, no dependencies. Hosted for free on GitHub Pages.

## Local preview

Open `index.html` in a browser, or serve it:

```sh
npx serve .
```

## Deployment

GitHub Pages serves the `main` branch root. Every push to `main` redeploys automatically.
